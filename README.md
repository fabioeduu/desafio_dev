# Desafio Dev - C#

Programa em C# com 3 desafios práticos.

## Os 3 Desafios

### 1. Comissões de Vendedores
Calcula comissão baseado no valor:
- Menos de R$100: sem comissão
- R$100 a R$499,99: 1%
- R$500 ou mais: 5%

### 2. Estoque
Registra entrada e saída de produtos com ID único.

### 3. Multa por Atraso
Calcula juros compostos: 2.5% ao dia.

## Como Usar

```powershell
git clone 
dotnet run
```

Ou clique duplo em `executar.bat`

## Arquivos

```
Desafio_dev/
├── Program.cs
├── Models/          # Classes de dados
├── Services/        # Lógica de cada desafio
├── Data/           # JSONs com dados
└── DesafioDev.csproj
```

## Menu

Ao executar, escolha:
- **1** = Comissões de vendedores
- **2** = Movimentações de estoque
- **3** = Cálculo de multa
- **0** = Sair

## Requisitos

- .NET 9.0 ou superior
- Windows/Linux/Mac
